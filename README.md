###### Using SASS Variables we can store 
- string  
- numbers
- colors
- booleans
- lists
- nulls


###### @mixin and @include 
- A mixin : a group of CSS declarations that can be reused throughout the style sheet.
- Ex: create Mixin: 
- @mixin para-style {
    font-size:16px;
    text-align:justify;}
- use mixin: 
- #about-me p{
    @include para-style;}
